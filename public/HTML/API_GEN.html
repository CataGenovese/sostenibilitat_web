API Gencat

Esta API de codigo abierto está basada en los incedios forestales de Cataluña en el año 2025, con detalle de la fecha, el municipio en el cuando se han iniciado y las superficies forestales y no forestales afectadas, en hectáreas.


Tipos de datos

FECHA INCENDIO: Fecha de inicio del incendio.

CÓDIGO COMARCA: Código ID de la comarca.

COMARCA: Comarca de inicio del incendio.

CÓDIGO MUNICIPIO: Código ID del municipio.

TERMEMUNIC: Municipio de inicio del incendio.

HAARBRADES: Superficie forestal arbolada quemadura, en hectáreas.

HANOARBRAD: Superficie forestal de matorral, herbácea o pasto quemado, en hectáreas.

HAFORESTAL: Superficie forestal quemada, en hectáreas. Suma la columna HaArbrades + HaNoArbrades.
