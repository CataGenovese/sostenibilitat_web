API Gencat

Esta API de codigo abierto está basada en los incedios forestales de Catalunya en el año 2025, con detalle de la fecha, el municipio en el cuando se han iniciado y las superficies forestales y no forestales afectadas, en hectáreas.

Campos disponibles

  • DATA INCENDI: Fecha de inicio del incendio.
  • CODI COMARCA: Código de comarca Idescat.
  • COMARCA: Nombre de la comarca.
  • CODI MUNICIPI: Código de municipio Idescat.
  • TERMEMUNIC: Municipio del incendio.
  • HAARBRADES: Hectáreas forestales arboladas.
  • HANOARBRAD: Hectáreas de matoll o herbáceas.
  • HAFORESTAL: Total forestal afectado (suma anterior).
